# Perch

**Clarity, presented beautifully.**

Perch is a Life Operating System that helps a person understand their life, finances, obligations, goals, opportunities, and projects at a moment's notice.

Perch is not just a dashboard. It is a calm place to see what is true, what matters, what changed, what needs attention, and what action is worth considering.

## Current Status

Perch is currently in early prototype and architecture development.

The repository contains:

- the Design Bible
- AI workspace files
- build plans
- product philosophy
- marketing concepts
- early project documentation

## Core Law

Perch never invents certainty.

Truth outranks convenience.

If Perch does not know something, it should say so.

## Project Goal

The first major product milestone is **Perch v0.5**: a small, trustworthy version Jeff can use daily.
