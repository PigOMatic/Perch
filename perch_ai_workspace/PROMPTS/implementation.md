# Implementation Prompt Pattern

Read:

- `perch_ai_workspace/README.md`
- `perch_ai_workspace/PROJECT_STATE.md`
- the relevant build plan

Implement only the requested task.

Do not broaden scope.

Preserve truth, source, and confidence rules.

When finished, report:

- files changed
- exact behavior changed
- behavior intentionally left unchanged
- tests or manual verification
- whether app code was modified
