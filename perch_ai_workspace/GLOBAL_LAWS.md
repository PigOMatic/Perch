# Global Laws

These laws govern Perch documentation, architecture, and implementation.

## Highest Law

Perch never invents certainty.

## Truth

- Truth outranks convenience.
- If Perch does not know, it says so.
- If Perch is unsure, it shows uncertainty.
- If two sources disagree, Perch exposes the disagreement.
- Every important claim should have source lineage.

## Engine Authority

The reasoning chain is:

```text
Truth Engine
    ↓
Priority Engine
    ↓
Recommendation Engine
    ↓
Voice Engine
    ↓
Pages
```

## Engine Responsibilities

- Truth answers: What is true?
- Priority answers: What deserves attention?
- Recommendation answers: What should the user do?
- Voice answers: How should this be said?
- Pages answer: How should this be presented?

## Page Rule

Pages present truth.

Pages do not invent facts, priority, recommendations, confidence, or certainty.

## Recommendation Rule

Recommendations must explain themselves.

Every meaningful recommendation should expose:

- action
- reason
- supporting evidence
- confidence
- source lineage
- reversibility when relevant
