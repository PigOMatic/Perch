# Object Assets

Reusable Perch object assets live here.

## Planned Object Families

- notebooks
- money envelopes
- sticky notes
- shift tickets
- journals
- clipboards
- maps
- mugs
- pens
- trail markers

## Live Data Rule

Object assets should be blank or decorative only. Text like amounts, due dates, reminders, and schedules must be rendered live by Flutter.
