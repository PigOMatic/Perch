# Home Perch Photoreal Asset Prompts

## Rules

- Generate photorealistic assets only.
- No text baked into images.
- Leave clean writable space where Flutter will place live text.
- Use consistent camera angle: 35-45 degree downward view.
- Use warm morning cabin light.
- Prefer WebP exports for app use.
- Transparent background for object assets when possible.
