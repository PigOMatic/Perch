# Time Branch Object

Status: layout experiment
Created: July 7, 2026

The Today story demo now tests a compact time-context object instead of a calendar strip.

The shape is a reverse branch:

- Today
- Work this week
- Next event

These three leaves feed into one central day-shape hub. The intent is to make time context fit in a tighter space and visually relate to the money marker tree.

This is not a full calendar. It is a time-context object for the Today page.

Design rule:

When time data is only meant to orient the user, show the few meaningful anchors instead of a mini calendar.
