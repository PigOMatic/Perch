# Today Run Sheet Environments

Status: active design direction
Created: July 7, 2026

Today is a one-minute relief page.

It is not a dashboard and it should not explain what to look at first. The layout should show importance through usefulness, placement, scale, and available actions.

The approved direction is a daily run sheet that can be viewed in different life environments.

## Three layers

1. Environment layer
   - Desk / coffee
   - Patio
   - Trail / field guide
   - Firepit
   - Break room
   - Kitchen table

2. Mood layer
   - Morning calm
   - Outside warmth
   - Evening firelight
   - Work-night focus
   - Family reset

3. Foreground information layer
   - Today
   - Next due
   - Money
   - Next shifts
   - Due soon
   - Calendar
   - Bills checked
   - From your brain

## Core rule

The same useful run sheet can sit in different places, but the information structure should stay familiar.

The setting gives emotional relief. The foreground gives clarity.

## Meaningful visual language

- Pinned paper = due item or user note
- Torn paper = lower-priority due-soon item
- Ticket = work shift
- Protected money surface = counted money and available choices
- Quiet pill = deeper view, such as Calendar or Bills checked
- Posted note = user-created reminder, concern, or task

Decoration should mean something. Random stickers should not become the design system.

## High-fidelity scene pass

The app should not stop at cards over a background. A mature Today screen should feel like a composed scene with information resting inside it.

For the desk environment, this means:

- Warm tabletop atmosphere
- Coffee/table cues in the outer environment
- A physical paper run sheet surface
- Layered notes, tickets, and soft shadows
- Organic placement without sacrificing readability
- Richer texture and depth than a plain app panel

The goal is picture-like charm while keeping components reusable.

## Today page feeling

The user should feel:

- I know what matters.
- I know what is safe.
- I know what is coming.
- I know what can wait.
- I can breathe.
