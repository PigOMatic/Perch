# Today Path Anchors

Status: layout experiment
Created: July 7, 2026

The Today story demo now tests a simpler path:

1. Today
2. Next work shifts
3. Next event

The goal is to make the page feel connected instead of stacked. The money marker tree remains the main action shape. Work shifts and the next event sit on the same visual path instead of separate dashboard cards.

This version removes the standalone nearby-context, people, and can-wait sections from the demo because they were not earning attention. Those details should return only when attached to the main path.

Design rule:

Nothing gets its own section unless it changes the meaning of the current marker.
