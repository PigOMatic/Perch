# Perch Layout Lab

**Status:** Initial design lab  
**Created:** July 5, 2026

---

## Purpose

`layout_lab.html` compares the same Today state across multiple Perch surface arrangements.

This exists so layout decisions happen before theme, color, texture, aging, stickers, and polish.

---

## Why this matters

Perch is not a single page design.

The same data and meaning must be able to appear on different surfaces while preserving the same layout anatomy:

```text
Orientation
Primary marker
Terrain/context
Margin notes
Trust/source
Next step
What can wait
```

## Evaluation questions

Use the lab to ask:

```text
Where does my eye go first?
Which surface feels most like Perch?
Which arrangement could scale to other pages?
Which one can support aging, stickers, tabs, and marks later?
Which one avoids dashboard energy?
```
