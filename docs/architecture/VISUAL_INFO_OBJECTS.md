# Visual Info Objects

**Status:** Design direction note  
**Created:** July 6, 2026

Perch should show more meaning with fewer long text blocks.

Useful compact objects:

```text
calendar squares
money chips
status badges
map markers
margin notes
person chips
source stamps
```

## Today example

The Today story demo now shows the next three days as small calendar squares:

```text
Sun 5 · Off
Mon 6 · Work · ICU 7p
Tue 7 · Recover
```

This makes schedule context easier to scan than a sentence.

## Rule

When information is repeated or structured, use a visual object before adding more prose.
