# Milestone 0: Home Perch

## Goal

A user opens Perch and immediately feels like they have arrived at a calm place where their life is already laid out.

## Current Build Slice

This slice adds the first experience pieces to the Flutter prototype:

- ambient animation controller
- drifting morning light
- coffee steam
- tappable notebook, envelope, sticky note, and shift tickets
- object focus overlay
- no dashboard navigation

## Definition of Done

Milestone 0 is complete when:

- [ ] Home Perch feels like a place, not an app screen
- [ ] notebook is the daily run sheet
- [ ] money envelope clearly communicates safety
- [ ] sticky note holds one brain item
- [ ] shifts feel like physical tickets
- [ ] coffee steam and light movement make the scene feel alive
- [ ] tapping objects feels natural
- [ ] production assets replace placeholder Flutter shapes
- [ ] scene runs on an actual device
- [ ] first 60 seconds are memorable

## Next Required Work

2. Add page-turn animation for notebook.
3. Add envelope opening animation.
4. Add sticky lift/flip animation.
5. Add first audio ambience hook.
